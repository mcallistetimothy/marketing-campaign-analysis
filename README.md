# Beauty Industry Consumer Insights

## Project Overview

This project analyzes 15,000 beauty products to identify factors influencing customer satisfaction, consumer engagement, and product performance.

Using Python, Pandas, and data visualization techniques, the analysis explores how pricing, ingredients, product categories, and cruelty-free positioning affect customer ratings.

---

## Business Questions

1. Which beauty brands receive the highest customer ratings?
2. Which product categories perform best?
3. Do higher-priced beauty products earn better ratings?
4. Do consumers prefer cruelty-free products?
5. Which ingredients appear most frequently in highly-rated products?
6. Which brands generate the highest levels of consumer engagement?

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab
- GitHub

---

## Key Findings

- Customer ratings remained relatively consistent across beauty product categories.
- Premium-priced products achieved only slightly higher ratings than lower-priced alternatives.
- Cruelty-free products received marginally higher ratings.
- Ingredients such as Hyaluronic Acid, Vitamin C, Retinol, Aloe Vera, and Shea Butter frequently appeared in highly-rated products.
- Several brands demonstrated exceptionally strong consumer engagement through high review volumes.

---

## Strategic Recommendations

- Prioritize ingredient effectiveness over premium pricing.
- Continue investing in ingredients associated with highly-rated products.
- Use cruelty-free positioning as a supporting value proposition.
- Analyze highly engaged brands to identify successful customer engagement strategies.
